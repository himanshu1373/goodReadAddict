search

Searches for a book based upon title and isbn#. Pagination is available by include values for the start index and maxResults.

Sample Call
{{docsSearchURL}}

Parameters

  • title: Title of the book
  • isbn: ISBN Number of book (Note: 13 digit and 10 digit ISBN numbers can both be used)
  • startIndex: index of first book to be returned (for use in pagination)
  • maxResults: The number of results to be fetched (default is “10” and max is “40”)
  • apikey[required]: Developer's api key

Response: 

{
	"restricted": (boolean) whether the user is restricted or not,
	"kind": (string) resource type for a volume,
	"totalItems": (int) total items returned by the search,
	"items": [
 		"kind": (string) resource type for a volume,
 		"id": (string) unique identifier for a volume,
 		"etag": (string) Opaque identifier for a specific version of a volume resource,
 		"selfLink": (string) Google Books API link to resource,
 		"volumeInfo": {
 			"title": (string) volume title,
 			"authors": (list) The names of the authors and/or editors for this volume,
 			"publisher": (string) Publisher of this volume,
 			"publishedDate": (string) Date of publication,
 			"industryIdentifiers": [
			{
				"type": (string) Identifier type. Possible values are ISBN_10, ISBN_13, ISSN and OTHER.,
				"identifer": (string) value of identifier.
			}],
			"readingModes":{
				"text": (boolean) whether text reading mode is available,
				"image": (boolean) whether image reading mode is available
			},
			"pageCount": (int) Total number of pages,
			"printType": (string) Type of publication of this volume. Possible values are BOOK or MAGAZINE
			"categories" : (list) A list of subject categories, such as "Fiction", "Suspense", etc.
			"averageRating": (double) The mean review rating for this volume (from Google Books API),
			"ratingsCount": (integer) The number of review ratings for this volume (from Google Books API),
			"contentVersion": (string) An identifier for the version of the volume content (text & images),
			"imageLinks": {} (list) A list of image links for all the sizes that are available,
			"language": (string) Best language for this volume (based on content). It is the two-letter ISO 639-1 code such as 'fr', 'en', etc,
			"previewLink": (string) URL to preview this volume on the Google Books site,
			"infoLink": (string) URL to view information about this volume on the Google Books site,
			"canonicalVolumeLink": (string) Canonical URL for a volume,
		}
 		"saleInfo": {
 			"country": (string) The two-letter ISO_3166-1 country code for which this sale information is valid. ,
 			"saleability": (string) Whether or not this book is available for sale or offered for free in the Google eBookstore for the country listed above. Possible values are FOR_SALE, FREE, NOT_FOR_SALE, or FOR_PREORDER.,
 			"isEbook": (boolean) Whether or not the volume is an ebook
 		}
 		"accessInfo": {
			"country": (string) The two-letter ISO_3166-1 country code for which this access information is valid.,
			"viewability": (string) The read access of a volume. Possible values are PARTIAL, ALL_PAGES, NO_PAGES or UNKNOWN. This value depends on the country listed above. A value of PARTIAL means that the publisher has allowed some portion of the volume to be viewed publicly, without purchase. This can apply to eBooks as well as non-eBooks. Public domain books will always have a value of ALL_PAGES.
			"embeddable": (boolean) Whether this volume can be embedded in a viewport using the Embedded Viewer API.	,
			"publicDomain": (boolean) Whether or not this book is public domain in the country listed above.	
			"textToSpeechPermission": (string) Whether text-to-speech is permitted for this volume. Values can be ALLOWED, ALLOWED_FOR_ACCESSIBILITY, or NOT_ALLOWED.,
			"epub": {
			  "isAvailable": (boolean) whether information about epub content is available,
			  "acsTokenLink": (string) URL to retrieve ACS token for epub download
			},
			"pdf": {
				"isAvailable": (boolean) whether information about pdf content is available,
			},
			"webReaderLink": URL to read this volume on the Google Books site,
			"accessViewStatus": (string) Combines the access and viewability of this volume into a single status field for this user. Values can be FULL_PURCHASED, FULL_PUBLIC_DOMAIN, SAMPLE or NONE,
			"quoteSharingAllowed": (boolean) Flag for quote sharing
		}
	  ]
}

API Tester

reviews/add

Adds a review for a particular book for a user

Sample Call
{{docsAddReviewsURL}}

Parameters

  • isbn[required]: ISBN Number of book (Note: 13 digit and 10 digit ISBN numbers can both be used)
  • bookId[required]: Unique ID of the book (from Google's API)
  • email[required]: email of user
  • review[required]: full text review by user
  • rating[required]: rating of book on scale of 1 to 5

Response 

{
  "totalReviews": count of reviews for this book,
  "reviews": [
    {
      "date": (string) date and time reviewed,
      "review": (string) full text review,
      "rating": (string) rating of book on scale of 1 to 5
      "user": (string) email of user
    }
  ],
  "success": (boolean) depending on success of operation
  
  Note: Also includes Google Books API info (See Search documentation)
}

API Tester

logs

Gets the api logs

Sample Call
{{docsGetLogsURL}}

Parameters (one of email, apikey is required)

  • email: email of user
  • apikey: Developer's api key
  • admin apikey[required]: Developer's api key (admin)

Response 
{
    "totalItems": (int) total logs returned,
    "logDate": (string) date logs being returned for,
    "items": [
        {
            "date": (string) date logs being returned for,
            "apiKey": (string) developer api key,
            "restricted": (string) n for not restricted, y for restricted,
            "userType": (string) limited or premium
            "operation": (string) description of api call by user
            "user": (string) email of user
        }
    ]
}

Note: Admin only

Every parameter listed is document based and the variable type doesn't count towards the descriptions
